THURSDAY - objects

Objects are basic building blocks. Next we'll build components to display in the browser, store data as objects, and retreive data on our file system.

VISUALLY Objects have curly braces, contain keys and values separated by a colon, and the properties (key/value pairs) are separated by a comma. Arrays have square brackets.

CONCEPTUALLY Arrays are collections of similiar things. Objects are ways to define details of a single thing. Both an array and an object can be assigned to a variable.

Never describe two things with a single object. Use an array to store two things inside of a single data structure.

Values can be integers, strings, booleans, even arrays. There are two ways to access values stored inside an object: dot notation and square bracket notation.

Dot Notation

console.log(`Obtain a value inside an object ${variableContainingTheObject.keyName}`);

Create a new key on an object - variableContainingTheObject.keyName = "value"

Square Bracket Notation

console.log(variableContainingTheObject["keyName"]);
